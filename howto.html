How to use epub-manga-creator? / 如何使用 epub-manga-creator

1. Click the file upload button. / 点击上传按钮

2. Select images (PNG. JPG. supported) to upload. / 选择图片（仅限png, jpg)

3. Fill in metadata of the manga. / 填入漫画基本信息

4. Fill in page size to which each image in its page will be stretched. / 填入单页大小，每个页面的图片将会被拉伸显示到这个比例的尺寸

5. Click the file download button and wait for a few seconds. / 点击下载按钮，等待数秒钟

6. Let's start reading! / 开始看吧
